Hi Guillaume, 

The JavaFront package that Eelco mentioned is hosted at https://github.com/metaborg/java-front. The spoofax branch includes a project set-up as a Spoofax project if interested. The java-bootstrap branch is easy to build with just Java tools:

$ wget http://hydra.nixos.org/job/strategoxt-java/strc-java-trunk/build/latest/download/2/strategoxt.jar
$ ANT_OPTS="-Xss8m -Xmx1g" ant -lib strategoxt.jar build install

You can then copy the things you need from the dist folder into your project.
